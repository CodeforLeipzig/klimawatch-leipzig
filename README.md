# klimawatch-leipzig
Analysis and visualizations of climate relevant data for the city of Leipzig 

## data
 * [Energie- und Klimaschutzprogramm (EKSP) 2030](https://ratsinformation.leipzig.de/allris_leipzig_public/vo020?VOLFDNR=2000269&refresh=false), see PDF attachments on the right side
 * [Statistik Leipzig](https://statistik.leipzig.de), e.g. population data
   * [OpenData API](https://statistik.leipzig.de/statserv/servod.aspx)
 * [Local Emission Framework](https://portal.emission-framework.org)
 * [Emissions API](https://emissions-api.org)
